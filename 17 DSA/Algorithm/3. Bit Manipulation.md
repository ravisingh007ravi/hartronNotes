## 🔷 What is Bit Manipulation?

Bit Manipulation me hum numbers ko **binary (0s & 1s)** me treat karke operations perform karte hain using bitwise operators.

👉 Simple line (Interview):

> “Bit manipulation uses binary representation of numbers to perform efficient operations using bitwise operators.”

---

## ⚙️ Basic Bitwise Operators

| Operator    | Symbol | Meaning         |              |
| ----------- | ------ | --------------- | ------------ |
| AND         | `&`    | dono bits 1 → 1 |              |
| OR          | `      | `               | koi ek 1 → 1 |
| XOR         | `^`    | different → 1   |              |
| NOT         | `~`    | invert bits     |              |
| Left Shift  | `<<`   | multiply by 2   |              |
| Right Shift | `>>`   | divide by 2     |              |

---

## 🧠 Important Properties

* `a ^ a = 0`
* `a ^ 0 = a`
* `a & 1` → check even/odd
* `a << 1` → multiply by 2
* `a >> 1` → divide by 2

## 🔁 Common Bit Tricks

### ✅ Check Even / Odd


---

### ✅ Swap without temp

```javascript id="y3yx5v"
a = a ^ b;
b = a ^ b;
a = a ^ b;
```

---

### ✅ Find Single Number

```javascript id="5bax1n"
const single = (nums) => {
    let res = 0;
    for (let num of nums) {
        res ^= num;
    }
    return res;
};
```

---

### ✅ Count Set Bits

```javascript id="2z8fvh"
const countBits = (n) => {
    let count = 0;
    while (n > 0) {
        count += (n & 1);
        n = n >> 1;
    }
    return count;
};
```

---

## 💻 Example (Addition without + operator)

```javascript id="o6t1dk"
const add = (a, b) => {
    while (b !== 0) {
        let carry = (a & b) << 1;
        a = a ^ b;
        b = carry;
    }
    return a;
};
```
