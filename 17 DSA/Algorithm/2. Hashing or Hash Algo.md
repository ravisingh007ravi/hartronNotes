# 📌 Hashing / Hash Map Algorithm – Complete Notes (DSA)

## 🔷 What is Hashing?

Hashing ek technique hai jisme hum **data ko key-value pair me store karte hain** taaki **fast lookup (O(1))** mil sake.

👉 Simple line (Interview):

> “Hashing is used to store and retrieve data in constant time using key-value mapping.”

---

## ⚙️ Key Concept

* **Key → Value mapping**
* Fast operations:

  * Insert → O(1)
  * Search → O(1)
  * Delete → O(1)

## 🔁 General Hashing Algorithm Steps

```text id="zj3b1a"
1. Empty hash map banao
2. Loop through data
3. Har element ko process karo
4. Check karo required value map me hai ya nahi
5. Agar hai → answer
6. Nahi hai → map me store karo
```

## 💻 Example (Two Sum – Hashing)

```javascript id="1r5n3c"
const twoSum = (nums, target) => {
    let map = {};

    for (let i = 0; i < nums.length; i++) {
        let diff = target - nums[i];

        if (map[diff] !== undefined) {
            return [map[diff], i];
        }

        map[nums[i]] = i;
    }

    return [-1, -1];
};
```

## 🧠 Example (Frequency Count)

```javascript id="teo0pn"
const countFreq = (arr) => {
    let map = {};

    for (let num of arr) {
        if (map[num]) map[num]++;
        else map[num] = 1;
    }

    return map;
};
```
